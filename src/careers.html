UI Artist - Unity

Remote
Full-time
Empire Games is venture backed start-up headquartered in London (United Kingdom). Building real money PvP casual skill games (eg. think player versus player solitaire or puzzle games) for the US and European markets. Founders have over 10 years experience in building successful mobile games, having led several products that reached over $200 million in yearly revenue.

Responsibilities

  • Produce stunning mobile UI artwork in collaboration with illustrators and designers.
  • Implement said artworks in Unity, utilising the tool to ensure UI will fit appropriately on all supported devices.
  • Animating UI within Unity, ensuring Unity Animator has the appropriate settings to switch between states.
  • Utilise particle systems, shaders and other tools to create beautiful visual effects.
